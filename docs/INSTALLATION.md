# Installation Guide

## Prerequisites
1. Disable Windows Defender Real-time Protection
2. Disable any antivirus software
3. Run as Administrator

## Installation Steps

### Step 1: Download
- Get latest release from GitHub
- Password for archive: `fortnite2024`

### Step 2: Extract
- Extract to any folder except Program Files
- Recommended: C:\Gaming\FortniteAdvantage

### Step 3: Configuration
- Edit configs/settings.json if needed
- Choose profile (Legit recommended)

### Step 4: Launch
1. Start Fortnite and get to lobby
2. Run FortniteAdvantage.exe as Admin
3. Wait for "Ready" message
4. Press INSERT in-game for menu

## Troubleshooting

### "Game not found"
- Make sure Fortnite is running
- Try running in Windowed Fullscreen

### "Injection failed"
- Disable antivirus
- Restart PC
- Try different injection method

### "Detected" message
- Update to latest version immediately
- Use HWID spoofer
- Contact support
